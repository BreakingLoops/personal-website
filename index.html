Getting Started with Ubuntu

I've been aiming to achieve better life-work balance for a while now, so today I decided to finally fix the laptop I bought a couple of months ago and install Ubuntu on it along with a suite of developer tools to keep me focus on the important things rather than simply shopping and Netflixing around. Yes, in my case life-work separation just meant loading up a different laptop, but what can I say? I tried...

What I've done:

  1. Download Ubuntu from official page
  2. Create bootable USB stick from ISO
  3. Set first booting device.
  4. Install Ubuntu
  5. Install VIM
  6. install Guake Terminal
  7. Install Git
  8. install Ruby
  9. install Homebrew
  10. install SublimeText
  11. install Chrome
  12. install SimpleNote (not required for coding, but helpful for writing down ideas as I go around)
